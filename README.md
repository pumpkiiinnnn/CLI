# CLI

В данной программе было реализовано два этапа:
1 этап. 
берёт параметры из командной строки 
проверяет, что обязательные есть
выводит в формате ключ=значение 
сохраняет всё в файл result.txt

2 этап.

программа принимает имя пакета и ссылку на файл. потом скачивает этот файл, распаковывает и проходит по всем пакетам. в функции extract_dependencies она находит нужный пакет и извлекает строку с зависимостями. потом зависимости очищаются от лишних симовлов и выводятся на экран в виде списка. 
если пакет не найен или зависимостей нет, то программа сообщает об этом
